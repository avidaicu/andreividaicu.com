NSW Design System

Reflecting on seven months

  • By Andrei Vidaicu
  • August 11, 2023

UX/UI

Components (Figma UI Kit)
  • Redesign of the Side navigation component (to support five levels of site hierarchy).
  • Tooltip/Toggletip component.
Templates (Figma UI Kit)
  • x3 Landing pages templates.
Audit (Figma UI Kit)

Staying consistent with the quality, identifying gaps, making sure everything is up to date by aligning the UI kit design assets with the DS documentation site.

Other

  • Collaborate closely with our design system partners, branding team, other designers to align with the brand guidelines ensure consistency, efficiency, and scalability in the design process.
  • Support and maintain the design system (Figma UI Kit).
  • Continuous design system engagement on the NSW Digital NSW community – used for collaborating, sharing and learning from the community.
  • Part of a team which participates at an enterprise-scale Agile SAFe framework, which involves 2 day PI (Planning Increment) events every 10 weeks.

In the pipelines

  • Moving retired versions of the design system into a separate file resulting in a better user experience.
  • Redesign of the ‘Using the design system’ section to improve on the flow of information and readability.
  • Component spec tool template to help ease design-to-engineering handoff (Figma UI Kit).
  • More and more collaborating and building relationships.
